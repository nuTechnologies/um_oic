# Open Xworks Design System

[![Matrix](https://img.shields.io/matrix/xwork%3Amatrix.org?logo=matrix)](https://app.element.io/#/room/#xwork:matrix.org)

The **xwork Design System** provides components and utilities for application and extension development within the
xwork Web ecosystem. It can be developed standalone via the design system documentation. The documentation is
built with [VitePress](https://vitepress.dev/).

Head over to the [hosted docs](https://docs.xwork.cloud/design-system/) for more information!

## Running the docs locally

To run the docs, you first need to install all dependencies:

```
pnpm i
```

Start docs in dev mode:

```
pnpm docs:dev
```

## Using the design-system as a developer

Please refer to the [docs](https://docs.xwork.cloud/design-system/gettingStarted/installation.html) for information about how to install and use the design-system as a developer.
